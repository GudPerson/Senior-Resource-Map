# CareAround SG User Guide

Last drafted: 2026-05-07

This guide is for guests, standard users, caregivers, and family members using CareAround SG to find senior-related support in Singapore.

It focuses on everyday use: finding resources, saving useful options, creating maps, sharing information, and keeping profile details up to date. Partner, admin, import, translation-review, and user-export workflows are intentionally left out of this public guide.

CareAround SG can help with planning, but it is not emergency, medical, financial, legal, or government-service advice. Before relying on a resource, check important details with the provider.

## Quick Terms

| Term | Meaning |
| --- | --- |
| Resource | Any place, programme, service, promotion, event, or support item listed in CareAround SG. |
| Place | A real-world location with an address, such as a centre, clinic, community club, or service office. |
| Programme/service | An activity, service, support item, event, or promotion. It may happen at one place or at several places. |
| Saved Resource | A resource you saved so you can find it again later. |
| My Directory | Your personal list of saved resources. |
| My Map | A map you create from saved resources. |
| Shared Map | A view-only map link that other people can open. |
| Linked place | A place connected to your account through a membership QR code or link. |
| Restricted offering | A programme or service that may depend on profile details, location, or linked membership. |

## 1. What You Can Do In CareAround SG

You can use CareAround SG to:

- Find senior-related places, programmes, services, promotions, and support.
- Search by keyword, service, tag, address, or postal code.
- Compare nearby resources in a list or on a map.
- Save useful resources to My Directory.
- Group saved resources into My Maps.
- Print a map, save it as an image, or share a view-only map link.
- Keep profile details up to date so results can be more relevant.
- Link a place membership when a QR code or membership link is provided.

Guests can browse public resources and open shared maps. To save resources, create maps, copy shared maps, or update your profile, sign in or create an account.

## 2. Choosing Language And Display Settings

CareAround SG supports English, Mandarin, Malay, and Tamil for many user-facing areas.

Use the language selector in the top navigation to change language. Some deeper operational, staff, legal, or admin material may stay in English until reviewed.

You can also use the accessibility controls in the top navigation to:

- Turn high contrast on or off.
- Increase text size.
- Decrease text size.

These settings are meant to make the app easier to read across cards, maps, and page surfaces.

## 3. Finding Resources In Discover

Open **Discover** to start browsing.

![Discover browse screen](/Users/sweetbuns/Documents/Senior-Resource-Map/docs/images/user-guide/discover-home.png)

You can search by:

- Resource name.
- Service or programme type.
- Tag.
- Address.
- Postal code.

You can also choose what to show:

- **All SG** to browse more broadly.
- **Places only** to focus on real-world locations.
- **Programmes & services only** to focus on activities, support, and services.
- **Saved only** to show resources you already saved.

![Discover filter panel](/Users/sweetbuns/Documents/Senior-Resource-Map/docs/images/user-guide/discover-filter-panel.png)

If you add a home postal code to your profile, CareAround SG can use it to show nearby resources and distance context. You may also search from a temporary postal code or use current location when your browser allows it.

On smaller screens, start with the browse list. Open the map when it is available and you want to compare where resources are located. If the map button is unavailable, continue browsing the list or adjust your location/search context first.

![Discover search results](/Users/sweetbuns/Documents/Senior-Resource-Map/docs/images/user-guide/discover-search-results.png)

## 4. Reading A Resource Page

Open a resource when you want to check details before saving, sharing, or visiting.

![Resource detail page](/Users/sweetbuns/Documents/Senior-Resource-Map/docs/images/user-guide/resource-detail.png)

A place page may show:

- Name.
- Address.
- Map location.
- Contact details.
- Opening hours.
- Website.
- Description.
- Related programmes or services.

A programme/service page may show:

- Name.
- Description.
- Schedule.
- Linked place or places.
- Contact details.
- Availability, if tracked.
- Eligibility or access notes, if restricted.

Before relying on a resource, check important details with the provider. This is especially important for opening hours, programme dates, fees, eligibility, registration steps, location, and availability.

## 5. Saving Resources

Saving a resource keeps it in My Directory so you can return to it later.

![Sign in page](/Users/sweetbuns/Documents/Senior-Resource-Map/docs/images/user-guide/sign-in.png)

To save resources:

1. Sign in or create an account.
2. Open Discover or a resource page.
3. Select the save button for the resource.
4. Open My Directory to see what you saved.

![Register tab](/Users/sweetbuns/Documents/Senior-Resource-Map/docs/images/user-guide/register.png)

Saving a resource does not register you for a programme or service. It only keeps the resource in your CareAround SG account.

If you remove a saved resource, it is removed from your personal list. It does not delete the resource from CareAround SG.

## 6. Using My Directory

My Directory is your personal saved-resource list.

Use it to:

- Review saved resources.
- Search saved resources by name, category, address, or type.
- Sort by recently saved, recently updated, or name.
- Remove resources you no longer need.
- Start creating maps from saved resources.

You may see status labels such as:

| Label | Meaning |
| --- | --- |
| No longer available | The live resource may have been removed, hidden, or changed. |
| Not shown on map | The resource is still listed, but it does not have a usable map location. |

These labels help you understand why a saved item may behave differently from other saved resources.

## 7. Creating And Managing My Maps

My Maps let you turn saved resources into a planning map.

A map can be useful when:

- Comparing support options near a home postal code.
- Preparing choices for a family discussion.
- Helping a caregiver or helper see several resources together.
- Grouping resources for a care situation.

To create a map:

1. Save a few resources from Discover or resource pages.
2. Open **My Directory**.
3. Open the **My Maps** tab.
4. Select **Create map**.
5. Choose the saved resources to include.
6. Give the map a clear name.
7. Create the map.

Inside a map, you can:

- Add or remove saved resources.
- Rename the map.
- Add or edit a description.
- Search within the map.
- Set a distance anchor, such as home or another postal code.
- View resources on the map or in the directory list.
- Print the map.
- Save the map as an image.
- Share or unshare a view-only link.

Removing a resource from a map removes it only from that map. The saved resource can still remain in My Directory.

## 8. Printing Or Saving A Map

Use print or image export when you need a copy for discussion or offline reference.

Before printing or saving a map:

- Check that the map title is clear.
- Check that the description explains what the map is for.
- Review resources that are not shown on the map.
- Check important provider details again if the map will be shared or used later.

Printed and saved copies may become outdated. If a programme date, fee, contact number, address, or eligibility rule matters, confirm it with the provider.

## 9. Sharing A Map

You can publish a view-only share link from your map.

When you share a map:

- Anyone with the link can open the shared map.
- Recipients can view the map and directory list.
- Changes you make to the original map may appear on the shared page.
- You can unpublish the link later.

Be careful before sharing a map that reveals personal planning information, such as where someone lives, which services they are considering, or what kind of support they may need.

To share a map:

1. Open your map.
2. Select **Share**.
3. Publish the share link.
4. Copy the link.
5. Send it only to intended recipients.

To stop sharing, open the share settings again and unpublish the link.

## 10. Viewing Or Copying A Shared Map

You can open a Shared Map link even without signing in.

As a guest, you can:

- View the shared map.
- Browse the directory list.
- Print the shared map.
- Open public resource details.

If you sign in, you may also be able to:

- Save individual resources from the shared map.
- Copy the shared map into your own My Maps.
- Personalise distance and nearby context with your own profile details.

Copying a shared map creates your own copy. It does not make you the owner of the original map.

## 11. Updating Your Profile

Open **My Profile** to keep your account details up to date.

Your profile can include:

- Name.
- Email address.
- Phone number.
- Postal code.
- Optional personalisation details such as date of birth, CHAS card, caregiver status, gender, property type, and volunteer interest.

Postal code is useful because it can help CareAround SG show nearby resources and services available in your service area.

Optional personalisation details help CareAround SG show programmes, services, and support that may be more relevant to you. They are used to help check whether some restricted offerings may be available. You can leave optional fields blank and update them later.

If you reached your profile from a restricted offering, return to that offering after saving your details to check the result again.

## 12. Verifying Your Phone, If Available

Your profile may show a phone verification panel.

If phone verification is available for your account:

1. Enter a valid Singapore mobile number.
2. Save your profile first.
3. Select **Verify with WhatsApp**.
4. Follow the WhatsApp verification step.
5. Keep the profile page open while CareAround SG checks the verification status.

Possible statuses include:

| Status | What it means |
| --- | --- |
| Save first | Save the phone number before verifying. |
| Not ready | Add a valid Singapore mobile number first. |
| Waiting | Verification has started; check WhatsApp and keep the page open. |
| Verified | The phone is linked to your CareAround account. |
| Try again | Verification did not complete or expired. |
| Needs help | The phone may already be linked to another account or needs manual review. |

If verification does not complete, try again when ready. If the panel says the phone needs help, contact CareAround support.

## 13. Linking Place Memberships

Some places may provide a membership QR code or link.

When you open a membership link:

1. CareAround SG checks the link.
2. If you are not signed in, you will be asked to sign in first.
3. After sign-in, CareAround SG links your account to the place.
4. The linked place appears in your profile.

A linked place may help member-only programmes or services recognise your access.

Membership linking does not automatically register you for every programme at that place. You should still check registration steps, eligibility, timing, and availability with the provider.

Review these pages when you want to understand how CareAround SG handles account information, browser preferences, and responsibilities:

## 15. Common Questions

### Do I need an account to browse?

No. You can browse public resources and open shared maps without signing in.

### Do I need an account to save resources?

Yes. Sign in or create an account to save resources, create maps, copy shared maps, and update your profile.

### Does saving a resource register me for a programme?

No. Saving only keeps the resource in My Directory. Contact or register with the provider if the programme requires it.

### Why is a saved resource not shown on the map?

It may not have a usable address or map location. It can still appear in the directory list.

### Why does a resource say it is no longer available?

The live resource may have been removed, hidden, or changed after you saved it. Check whether another current resource is available.

### Why am I asked for a postal code?

Postal code helps CareAround SG show nearby resources and services for your area. You can add or update it in your profile.

### Why am I asked for optional profile details?

Optional details can help personalise recommendations and check whether restricted offerings may be available to you. They are optional unless a specific restricted offering needs them.

### Can I stop sharing a map?

Yes. Open the map, select sharing settings, and unpublish the share link.

### Can someone edit my map through a shared link?

No. Shared maps are view-only. A signed-in recipient may be able to copy the map into their own My Maps.

### What should I check before relying on a resource?

Check the provider's latest information for opening hours, programme dates, registration steps, fees, eligibility, address, contact number, and availability.

## 16. Short Practice Walkthroughs

### Find And Save Resources

1. Open Discover.
2. Search by postal code or keyword.
3. Open a resource.
4. Check the address, schedule, contact details, and description.
5. Save the resource.
6. Open My Directory to see it.

### Create A Care Planning Map

1. Save several useful resources.
2. Open My Directory.
3. Open My Maps.
4. Create a map.
5. Choose the saved resources to include.
6. Add a clear map name and description.
7. Set a distance anchor if useful.
8. Print, save as image, or share only when the details look right.

### Use A Shared Map

1. Open the shared map link.
2. Browse the map and directory list.
3. Sign in if you want to save resources or keep your own copy.
4. Check provider details before relying on the information.
